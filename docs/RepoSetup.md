# Repository Setup

## GitHub

Install from GitHub.

```bash
$ git clone https://github.com/lehcode/soakp.git
```

## NPM

Install as NPM package to use as a dependency in your project.

```bash
$ npm install @lehcode/soakp
```

See [Run with NPM](./usage/NodejsLibrary.md) section.
